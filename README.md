# E-Commerce Laptop Store 

A functional E-Commerce prototype built with PHP (Procedural), MYSQL, and JavaScript.


## Features
- **Frontend:** HTML5, CSS Grid/Flexbox, Responsive Design.
- **Backend:** PHP with MySQLi (Prepared Statements).
- **Database:** Relational Schema with Users, Orders, and Order Items.
- **Key Functionality:**
    - Product Listing & Details.
    - Session-based Shopping Cart (Add/Remove).
    - Guest Checkout with SQL Transaction Safety.
    - Inventory Management logic.


## Setup
1. Import `ecommerce_db` using the provided SQL Scripts.
2. Configure `db_connect.php`.
3. Run on Apache (XAMPP/WAMP).
